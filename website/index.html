Get Started in 30 Seconds

Install, import, and start using TONL

1. Install

$ npm install tonl

2. Import

import { TONLDocument } from 'tonl';

3. Use

// Create document
const doc = TONLDocument.fromJSON({
  users: [
    { name: 'Alice', age: 30, role: 'admin' },
    { name: 'Bob', age: 25, role: 'user' }
  ]
});

// Query data
const admins = doc.query('users[?(@.role == "admin")]');

// Modify data
doc.set('users[0].age', 31);

// Save to file
await doc.save('data.tonl');

That's it! You're using TONL!

Query, modify, and save your data with a powerful API - all while saving 32-45% on storage and LLM costs.
